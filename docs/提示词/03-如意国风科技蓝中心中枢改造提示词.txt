推荐实现方向：

优先使用 ECharts effectScatter + lines + cartesian2d。

实现建议：

- coordinateSystem 使用 cartesian2d。
- xAxis / yAxis 隐藏，范围 0 到 100。
- 中心节点坐标为 [50, 50]。
- 周围节点使用 0-100 抽象坐标环绕中心。
- lines 表示数据链路，可以开启流光 effect。
- effectScatter 表示周围业务节点。
- scatter 表示中心中枢节点。
- 可叠加一个或多个无坐标轴辅助 series，营造环形光晕或背景纹理。
- 保留 observeResize 逻辑，确保图表随容器 resize。

可选实现方向：

也可以使用 ECharts graph：

- layout 使用 none。
- 节点固定 x / y 坐标。
- 中心节点在 x=50, y=50。
- 周围节点按环形分布。
- links 表示数据链路。
- edgeSymbol 可以使用 ['none', 'arrow']。
- lineStyle 使用半透明青蓝或玉青。
- emphasis 开启邻接高亮。

如果选择 graph，请确保视觉效果足够像大屏主视觉，而不是普通关系图。
